# NAME

**playonbsd-cli** - manage games on OpenBSD

# SYNOPSIS

playonbsd-cli
\[]
\[]
\[]
\[]
\-hv-manMODE...

# OPTIONS

**-h/--help**

> This message.

**-v/--version**

> Print version string.

# DESCRIPTION

**This program**
provides several modes to set up and run games, and manage your game
library.
